# E-Guro LMS Redesign

This project is a redesign of the **E-Guro LMS** (Learning Management System) for **City College of Calamba**. The focus was to improve the aesthetics and functionality of the **login page**.

## Technologies Used
- **HTML**
- **CSS**
- **JavaScript**
